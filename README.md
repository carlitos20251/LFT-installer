# Sobre LFT
LFT es una especie de 'distribucion', o mejor dicho instalador,este instalador instala arch linux,gentoo y debian en sus verciones bootstrap de forma sencilla
## Instalacion
La instalacion es tan facil como correr:
```
git clone https://github.com/carlitos20251/LFT-installer.git
cd LFT-installer
chmod +x install.sh
chmod +x distros/*
./install.sh
```
